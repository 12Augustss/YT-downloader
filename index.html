YouTube loader สำหรับวิดีโอที่ได้รับอนุญาตเท่านั้น

โหลดวิดีโอแบบปลอดภัยผ่าน Backend ของคุณ

หน้าเว็บนี้ใช้เป็นหน้า Frontend สำหรับรับลิงก์ YouTube แล้วส่งไปยัง API/Backend ที่คุณตั้งเอง เช่น Cloudflare Worker, Render, VPS หรือเครื่อง local ที่รัน yt-dlp

ใช้ได้เฉพาะวิดีโอที่คุณเป็นเจ้าของ, ได้รับอนุญาต, หรือมีสิทธิ์ดาวน์โหลดเท่านั้น หน้าเว็บนี้ไม่ฝัง token ลับ และไม่ bypass ระบบป้องกันของแพลตฟอร์ม 

        

      


      

        
วิธีต่อให้ใช้งานจริง

        

          
1เอาไฟล์นี้ไปวางเป็น index.html ใน GitHub Pages ได้เลย

          
2สร้าง backend ที่มี endpoint /download เพื่อรับ URL แล้วรัน yt-dlp ฝั่ง server

          
3ตั้งค่า API Endpoint ด้านล่าง เช่น https://your-api.com

          
4เมื่อ backend โหลดเสร็จ ให้ตอบกลับเป็นลิงก์ไฟล์สำหรับดาวน์โหลด
